4、前端初步了解

1）什么是前端开发工程师

前端工程师是互联网时代软件产品研发中不可缺少的一种专业研发角色。
从狭义上讲，前端工程师使用HTML、CSS、JavaScript等专业技能和工具将产品UI设计稿实现成网站产品，涵盖用户PC端、移动端网页，处理视觉和交互问题。
从广义上来讲，所有用户终端产品与视觉和交互有关的部分，都是前端工程师的专业领域。现在的W3C标准不仅仅局限于浏览器，还包括各种手持智能设备，车载设备、智能家居等等。

在未来万物互联的时代，前端将不仅仅是网页上的工程师， 而是所有人机交互领域的工程师。
凡是我们肉眼所看见的页面、样式、文字、图片等以及人和机器之间的一些交互动作等基本上都是前端开发完成的

6、什么是语言

人与人之间的沟通交流，我们会用到各种语言，比如：汉语，英语… ；
作为一个前端工程师，要与浏览器沟通交流，就要用到浏览器所能识别的语言；

  • HTML结构层

    网页的骨架，HTML是网页内容的载体，内容就是网页制作者放在页面上想要让用户浏览的信息，可以包含文字、图片、视频等。

  • CSS 样式层

    CSS样式是表现。就像网页的外衣，比如，标题字体、颜色变化，或为标题加入背景图片、边框等，所有这些用来改变内容外观的东西称之为表现。

  • javascript 行为层/交互层

    JavaScript是用来实现网页上的特效效果。如：鼠标滑过弹出下拉菜单。或鼠标滑过表格的背景颜色改变。还有焦点新闻（新闻图片）的轮换。可以这么理解，有动画的，有交互的一般都是用JavaScript来实现的。

8、HTML简介

1）什么是HTML

  • HTML指的是超文本标记语言(HyperText Markup Language)，

  • HTML不是一种编程语言，而是一种标记语言(markup language)

  • 标记语言是一套标记标签(markup tag)

  • HTML文档包含了HTML标签及文本内容

  • HTML文档也叫做web页面

HTML文件的特点

浏览器根据标记来解析和描述内容（从html的第一行开始向下解析），如果遇到错误，浏览器不会指出，也不会报错，会跳过此处继续加载，编辑人员需要根据显示效果去分析哪里出错误了 – html不会报错

html的兼容问题

因为浏览器是由不同厂商生产的，所以对html的解析有分歧，产生了页面效果显示不一致的情况，我们将这个问题，叫做兼容问题。

2）HTML标签

HTML标记标签通常被称为HTML标签，HTML标签是HTML语言中最基本的单位

  • 分类

    • 开始标签<html>

    • 结束标签 </html>

    • 空标签（单标签）<meta />

  • 特点

    • 由尖括号包围的关键词，例 <html>

    • 通常是成对出现的，例 <div> 和 </div>

    • 标签对中的第一个标签是开始标签，第二个标签是结束标签

    • 开始和结束标签也被称为开放标签和闭合标签。

    • 也有单独呈现的标签，如：等。

3）HTML元素

HTML元素通常由开始标签、结束标签和元素内容三部分组成

  • 语法：<div>这是div元素</div>

  • HTML元素分类

    • HTML 常规元素

      • 由开始标签和结束标签组成，标签之间可以进行嵌套

    • HTML 空元素（空元素在开始标签中进行关闭）

      • 只有一个空标签，不能进行嵌套

      • <img src="图片路径" />

注：”HTML 标签” 和 “HTML 元素” 通常都是描述同样的意思.

4）HTML标签属性

属性是HTML元素提供的附加信息

1、属性书写规范

  • 属性一般在开始标签中书写

  • 组成：标签属性名和标签属性值。name=”value”。

  • 语法：标签属性名=”标签属性值”

    • 标签属性值要用""双引号或者”包起来

  • 属性值建议使用小写

    • 建议使用小写属性

    • 万维网联盟在其 HTML 4 推荐标准中推荐小写的属性/属性值。

    • 而新版本的 (X)HTML 要求使用小写属性

  • HTML 属性参考手册

    • 标准属性参考手册 http://www.w3school.com.cn/tags/html_ref_standardattributes.asp
